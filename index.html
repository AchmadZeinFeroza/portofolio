Profil

Saya juga seorang freelance webdeveloper, kesukaan saya terhadap website membuat saya senang dan dapat menikmati pekerjaan ini.

Sistem Deteksi Hama dan Penyakit Sawi

Tugas Akhir Perkuliahan

Full Stack Web Develeper

Sistem Deteksi Hama dan Penyakit Sawi merupakan project tugas akhir perkuliahan saya. Sistem ini dibuat untuk membantu para petani dalam mengatasi hama dan penyakit pada tanaman Sawi. Sistem dibuat menggunakan metode Convolutional Neural Network (CNN) dengan model arsitektur Mobilenet Versi 2. Pembuatan sistem ini menggunakan framework Laravel dan framework Pytorch (salah satu framework Python), serta menggunakan API untuk menghubungkan kedua bahasa pemrograman tersebut.
